W3C - Metadata

  • head

    Representa uma coleção de metadados para o Documento.

    
<!doctype html>
<html>
    <head>
        <title>A document with a short head</title>
    </head>
<body>

  • title

    • Contexto: Dentro do head, não contendo nenhum outro elemento 
      <title>
    • Representa o nome ou título do documento.
    • Atributos
      • Atributos Globais

    <title></title>

  • base

    • Contexto: Dentro da tag head, não contendo nenhum outro elemento <base>
    • Atributos:
      • Atributos Globais
      • href
      • target
    • Endereço base para resolução de links relativos na página.

    
<-- <base > -->
<!DOCTYPE html>
<html>
    <head>
        <title>This is an example for the <base> element</title>
        <base href="http://www.example.com/news/index.html">
    </head>
    <body>
        <p>Visit the <a href="archives.html">archives</a>.</p>
    </body>
</html>

  • link

    • Contexto: Dentro da tag head
    • Cria link para outro recurso dentro do documento.
    • Sumário para Links
    • Atributos:
      • Atributos Globais
      • href
      • crossorigin
      • rel
      • media
      • hreflang
      • type
      • sizes

                                <link rel=alternate href="/en/html" hreflang=en type=text/html title="English HTML">
<link rel=alternate href="/fr/html" hreflang=fr type=text/html title="French HTML">
<link rel=alternate href="/en/html/print" hreflang=en type=text/html media=print title="English HTML (for printing)">
<link rel=alternate href="/fr/html/print" hreflang=fr type=text/html media=print title="French HTML (for printing)">
<link rel=alternate href="/en/pdf" hreflang=en type=application/pdf title="English PDF">
<link rel=alternate href="/fr/pdf" hreflang=fr type=application/pdf title="French PDF">

<!-- a persistent style sheet -->
<link rel="stylesheet" href="default.css">

<!-- the preferred alternate style sheet -->
<link rel="stylesheet" href="green.css" title="Green styles">

<!-- some alternate style sheets -->
<link rel="alternate stylesheet" href="contrast.css" title="High contrast">
<link rel="alternate stylesheet" href="big.css" title="Big fonts">
<link rel="alternate stylesheet" href="wide.css" title="Wide screen">

  • meta

    • Atributos:
      • Atributos Globais
      • name
      • http-equiv
      • content
      • charset
    • Representa vários tipos de metadados que não podem sem expressados usando os elementos, title, base, link, style e script. 

                                <meta charset="utf-8">

<meta http-equiv="Refresh" content="300">

<meta http-equiv="Refresh" content="20; URL=page4.html">

<meta name="keywords" content="british,type face,font,fonts,highway,highways">

  • style

    • Atributos:
      • Atributos Globais
      • media
      • type
    • Permite a inserção de estilo, código CSS, incorporado, embedded no documento.

                                <style>
   body { color: black; background: white; }
   em { font-style: normal; color: red; }
</style>

W3C - Grouping Content

  • Elemento 
    <p></p>
    • Representa um parágrafo.
  • Elemento 
    <hr />
    • Representa uma quebra temática no nível de parágrafo. Ex: mudança de cena em uma estória, transição de tópico, etc.
  • Elemento 
    <pre></pre>
    • Representa um bloco de texto preformatado.
  • Elemento 
    <blockquote></blockquote>
    • Representa conteúdo citado de outra fonte.
  • Elemento 
    <ol></ol>
    • Representa lista de items ordenados.
  • Elemento 
    <ul></ul>
    • Representa lista de items não ordenados.
  • Elemento 
    <li></li>
    • Representa um item de uma lista.
  • Elemento 
    <dl></dl>
    • Representa uma lista de associação, lista de descrição, consituída de elementos <dt>(nomes) e <dd>(valores).
  • Elemento 
    <dt></dt>
    • Representa elementos de nomes em listas de associação <dl>;
  • Elemento 
    <dd></dd>
    • Representa elementos de valores em listas de associação <dl>;
  • Elemento 
    <figure></figure>
    • Um tipo de “flow content” com caption, legenda.
  • Elemento 
    <figcaption></figcaption>
    • Representa uma legenda para uma figura.
  • Elemento 
    <div></div>

W3C - Text-Level Semantics

  • Elemento 
    <a></a>
    • Representa um hyperlink.
  • Elemento 
    <strong></strong>
    • Representa um conteúdo importante, urgente.
  • Elemento 
    <i></i>
    • Indica uma qualidade de texto diferente do text normal, antigo italic;
  • Elemento 
    <b></b>
    • Indica um trecho de texto que requer atenção. bold.
  • Elemento 
    <span></span>
    • Sem significado semântico, pode ser útil sendo utilizado com atributos globais como class, ou id.
  • Elemento 
    <br/>
    • Representa uma quebra de linha.

W3C - Tabular Data

  • Elemento 
    <table></table>
    • Representa uma tabela.
  • Elemento 
    <caption></caption>
    • Representa o título da tabela.
  • Elemento 
    <colgroup></colgroup>
    • Representa um grupo de uma ou mais colunas em uma tabela.
  • Elemento 
    <col></col>
    • Representa uma coluna em um colgroup.
  • Elemento 
    <tbody></tbody>
    • Representa um bloco de linhas que consiste em um bloco de dados da tabela.
  • Elemento 
    <thead></thead>
    • Representa um bloco de linhas que consiste em um cabeçalho da tabela.
  • Elemento 
    <tfoot></tfoot>
    • Representa um bloco de linhas que consiste em um sumário da tabela.
  • Elemento 
    <tr></tr>
    • Representa uma linha de células na tabela.
  • Elemento 
    <td></td>
    • Representa uma célula na tabela.
  • Elemento 
    <th></th>
    • Representa uma célula de cabeçalho.
    • Atributos
      • scope - Specifies which cells the header cell applies to
      • abbr - Alternative label to use for the header cell when referencing the cell in other contexts
      • sorted - Column sort direction and ordinality
    • Atributos <th> e <td>
      • colspan - Number of columns that the cell is to span
      • rowspan - Number of rows that the cell is to span
      • headers - The headers for this cell

W3C - Forms

  • Elemento 
    <form></form>
    • Representa uma coleção de elementos de formulário, onde alguns dos elementos podem representar valores editáveis e podem ser submetidos ao servidor para processamento.
    • Atributos
      • accept-charset - Character encodings to use for form submission
      • action - URL to use for form submission
      • autocomplete - Default setting for autofill feature for controls in the form
      • enctype - Form data set encoding type to use for form submission
      • method - HTTP method to use for form submission
      • name - Name of form to use in the document.forms API
      • novalidate - Bypass form control validation for form submission
      • target - Browsing context for form submission
  • Elemento 
    <input />
    • Representa um campo de dados
    • Atributos
      • accept - Hint for expected file type in file upload controls
      • alt - Replacement text for use when images are not available
      • autocomplete - Hint for form autofill feature
      • autofocus - Automatically focus the form control when the page is loaded
      • checked - Whether the command or control is checked
      • dirname - Name of form field to use for sending the element's directionality in form submission
      • disabled - Whether the form control is disabled
      • form - Associates the control with a form element
      • formaction - URL to use for form submission
      • formenctype - Form data set encoding type to use for form submission
      • formmethod - HTTP method to use for form submission
      • formnovalidate - Bypass form control validation for form submission
      • formtarget - Browsing context for form submission
      • height - Vertical dimension
      • inputmode - Hint for selecting an input modality
      • list - List of autocomplete options
      • max - Maximum value
      • maxlength - Maximum length of value
      • min - Minimum value
      • minlength - Minimum length of value
      • multiple - Whether to allow multiple values
      • name - Name of form control to use for form submission and in the form.elements API
      • pattern - Pattern to be matched by the form control's value
      • placeholder - User-visible label to be placed within the form control
      • readonly - Whether to allow the value to be edited by the user
      • required - Whether the control is required for form submission
      • size - Size of the control
      • src - Address of the resource
      • step - Granularity to be matched by the form control's value
      • type - Type of form control
      • value - Value of the form control
      • width - Horizontal dimension
      • Also, the title attribute has special semantics on this element when used in conjunction with the pattern attribute.

  • Elemento 
    <button />
    • Representa um botão
    • Atributos
      • autofocus - Automatically focus the form control when the page is loaded
      • disabled - Whether the form control is disabled
      • form - Associates the control with a form element
      • formaction - URL to use for form submission
      • formenctype - Form data set encoding type to use for form submission
      • formmethod - HTTP method to use for form submission
      • formnovalidate - Bypass form control validation for form submission
      • formtarget - Browsing context for form submission
      • menu - Specifies the element's designated pop-up menu
      • name - Name of form control to use for form submission and in the form.elements API
      • type - Type of button
      • value - Value to be used for form submission
    • Types
      Keyword State Brief description
      submit Submit Button Submits the form.
      reset Reset Button Resets the form.
      button Button Does nothing.
  • Elemento

  • Elemento

  • Elemento 
    <textarea>
    • Representa um controle de edição multlinha.
  • Elemento 
    <keygen>
    • Representa um controle para geração de key par.
  • Elemento 
    <output>
    • Representa um resultado de um cálculo ou de uma ação do usuário.
  • Elemento 
    <progress>
    • Representa uma programa de progresso.
  • Elemento 
    <meter>
    • Representa uma medida de escalar com uma extensão, um range, ou valor fracional conhecido. Ex: Uso de disco, frações de votos entre candidatos, etc.

    • Representa uma imagem.
  • Elemento
